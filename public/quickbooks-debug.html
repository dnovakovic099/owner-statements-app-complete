QuickBooks Debug Helper

This page will help you manually get the QuickBooks tokens step by step.

Step 1: Get Your Authorization URL

What to do:

  1. Copy the URL above
  2. Open it in a new browser tab
  3. Complete the QuickBooks authorization
  4. When redirected back, you'll see an error page - that's normal!
  5. Look at the URL in your browser address bar

Step 2: Extract Values from Callback URL

After authorization, you'll be redirected to a URL that looks like this:

http://localhost:3003/api/quickbooks/auth/callback?code=AB11234567890&realmId=9130359982560895033&state=xyz123

Extract these values:

  • code = The authorization code (starts with "AB")
  • realmId = Your QuickBooks Company ID (long number)

Step 3: Exchange Code for Tokens

Use this curl command to get your access tokens:

curl -X POST "http://localhost:3003/api/quickbooks/auth/callback" \ -H "Content-Type: application/json" \ -H "Authorization: Basic $(echo -n 'LL:bnb547!' | base64)" \ -d '{ "code": "PASTE_YOUR_CODE_HERE", "realmId": "PASTE_YOUR_REALM_ID_HERE" }'

Replace:

  • PASTE_YOUR_CODE_HERE with the code from step 2
  • PASTE_YOUR_REALM_ID_HERE with the realmId from step 2

Step 4: Use Manual Setup Page

Once you have the tokens from step 3, use the manual setup page:

Go to Manual Setup Page
Important Notes:
  • The callback URL will show an error - ignore it, just look at the URL parameters
  • The authorization code can only be used once
  • If something goes wrong, start over from step 1
